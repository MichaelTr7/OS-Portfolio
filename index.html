Projects

Mult-e-Tool

The streamline electronics tool.

  • Built using HTML, CSS and JavaScript.
  • Colour codes to resistance.
  • Capacitor codes to capacitance and vice-versa.

Icon Converter

A Windows and macOS app icon generator.

  • Converts PNG images into macOS (.icns) and Windows (.ico) icon formats.
  • Built using HTML, CSS, JavaScript, Bash and Electron framework.

Fuse & Sound Module

Regulates 12V input to 9V and 5V rails.

  • Square LEDs indicating active rails.
  • Speaking plays a two second tone to indicate powering on sequence.

LED Spectrum Analyzer

Visualizes frequencies contained in music.

  • Visulizes six frequency bands.
  • Drives 336 LEDs and dual layer PCBs.
  • Tunable sensitivity through potentionmeter knobs.

GPA Grapher

A simple grade point grapher.

  • Calculates and graphs CGPAs.
  • Generates PI chart breakdown.
  • Organizes marks by semester and compares CGPA vs Term GPA.

Folder Painter

Customizable folders for macOS.

  • Hue transformation accomplished using OpenCV and Python.
  • NodeJS subprocess module used for joining web and Python code.

Audio Dock

Audio visualization using microphone.

  • Supports 5 different visulaization styles.
  • Transparent backgrounds allows visulaization to seamlessly overlay.
  • Powered by P5.js.

Routine Square

Interactive checklist of 5 key daily tasks.

  • Interactive checkmarks.
  • Opens in upper right of screen.
  • Automatically resets checkmarks at midnight.

ImTune

Adjusting image hue mathematically.

  • Built programatically using MATLAB.
  • Takes any image file and allows the hue to be calibrated through the adjustement knob.

Simple Connect4

The classic game of Connect4.

  • Web implementation of a traditional Connect4 game.
  • Sound effects generated using GarageBand.

Arduino Mega Shield

Routes pins to screw terminal blocks.

  • Designed and sent to fabrication.
  • Used for final year capstone project.
  • Allows for secure wiring and easier prototyping.

Lithium-Ion Battery Tester

Culminating capstone project.

  • Collaboratively worked on with three other teammates.
  • Successfully made a modular prototype supporting four 18650 batteries.
